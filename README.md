## 内容简介

本手册详细介绍了 `mathmodels` 包中实现的各种数学建模算法，目前只实现评价类算法。内容覆盖了从**数据预处理**、**多种客观赋权方法**（熵权法、CRITIC、主成分）、**主客观组合赋权**，到**综合评价技术**（TOPSIS、灰色关联分析、秩和比法、模糊综合评价、DEA）、**不平等度量**（基尼系数、泰尔指数）以及**系统评价**（耦合协调度、障碍度）等。

本书与包的核心特色在于**算法严谨**、**接口易用**，并与 `tidyverse` 生态无缝集成，旨在简化复杂的数学建模过程。
